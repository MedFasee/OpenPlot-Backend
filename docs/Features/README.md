# Features do OpenPlot.Api

## Visăo Geral

Esta pasta reúne a documentaçăo técnica das features expostas pelo projeto `OpenPlot.Api`.

## Índice

- [`runsEndpoints.md`](runsEndpoints.md) — feature de consultas de runs, terminais e séries temporais.
- [`authEndpoints.md`](authEndpoints.md) — autenticaçăo e encerramento de sessăo.
- [`searchEndpoints.md`](searchEndpoints.md) — criaçăo, compartilhamento, exclusăo lógica e consulta de buscas.
- [`configEndpoints.md`](configEndpoints.md) — catálogo de PDCs e terminais/PMUs.
- [`importEndpoints.md`](importEndpoints.md) — importaçăo de arquivos XML para carga de metadados.
- [`postProcessingEndpoints.md`](postProcessingEndpoints.md) — pós-processamento baseado em `cache_id`.

## Diretriz

Todos os documentos desta pasta seguem o mesmo padrăo:

- visăo geral da feature;
- responsabilidades técnicas;
- endpoints expostos;
- entradas e saídas principais;
- dependęncias e observaçőes de implementaçăo.
